## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone [https://github.com/CPSG-26/qalbu.git](https://github.com/degalih/backend-personal-finance-tracker)
   ```
2. Run Composer Install
   ```sh
   composer install
   ```
3. Start Development Server
   ```sh
   php artisan serve
   ```
